Stracture :

<!DOCTYPE html>
To tell the browser this file is "HTML5" version.
<html></html>
All HTML code will write inside this main tag.
<head></head>
It contains all information about the web-page.
Examples:
  • <meta> It defines metadata about an HTML document.
  • <title></title> It contains the title of the web-page.
  • <link> It contains the link of any external file.
<body></body>
It contains all the content of the web-page.
Examples:
  • <h1> It is used to define a head "most important one".
  • <p> It is used to define a paragraph of text.
  • <a> It is used to define a link.
  • <img> It is used to define an image.

Elements in HTML

The HTML element is everything from the start tag to the end tag: <tagname>Content goes here...</tagname>

Examples of some HTML elements:
  • The <html> element is the root element and it defines the whole HTML document.
  • The <body> element defines the document's body.
  • The <h1> element defines a heading.
  • The <p> element defines a paragraph.
  • The <br> tag defines a line break, and is an empty element without a closing tag

Attributes in HTML

  • All HTML elements can have attributes.
  • Attributes provide additional information about elements
  • Attributes usually come in name/value pairs like: name="value"

Global Attributes:

The global attributes are attributes that can be used with all HTML elements.

Examples:

Attribute Description
class Specifies one or more classnames for an element (refers to a class in a style sheet).
id Specifies a unique id for an element.
hidden Specifies that an element is not yet, or is no longer, relevant.

Elements Attributes:

The elements attributes provide additional information about elements.

Examples:

Attribute Belongs to Description
alt <img> Specifies the types of files that can be accepted by file upload element.
charset <meta> Specifies the character encoding for the HTML document.
href <a> , <link> Specifies the URL of the page the link goes to.

HTML Formatting Elements

Formatting elements were designed to display special types of text:

  • <b> - Bold text => This text is bold.
  • <strong> - Important text => This text is important!
  • <i> - Italic text => This text is italic
  • <em> - Emphasized text => This text is emphasized
  • <mark> - Marked text => This text is marked text
  • <small> - Smaller text => This is some smaller text.
  • <del> - Deleted text => This text is deleted text
  • <ins> - Inserted text => My favorite color is blue red
  • <sub> - Subscript text => This is subscripted text
  • <sup> - Superscript text => This is superscripted text.

HTML Links

  • HTML links are hyperlinks.
  • You can click on a link and jump to another document.
  • The HTML <a> tag defines a hyperlink.

HTML Links - Syntax

  • <a href="url"> link text </a>
  • The most important attribute of the <a> element is the href attribute, which indicates the link's destination.
    • Use "mailto:" inside the href attribute to create a link that opens the user's email program
  • The target attribute specifies where to open the linked document.
    • The target attribute can have one of the following values:
    • _self - Default. Opens the document in the same window/tab as it was clicked
    • _blank - Opens the document in a new window or tab
    • _parent - Opens the document in the parent frame
    • _top - Opens the document in the full body of the window

Absolute URLs vs. Relative URLs

  • Using an absolute URL (a full web address) in the href attribute.
    • <a href="https://www.google.com/"> Google </a>
  • A local link (a link to a page within the same website) is specified with a relative URL (without the "https://www" part)
    • <a href="/css/default.asp"> CSS Tutorial </a>

HTML Images

  • Images are used to add graphics, charts, and pictures to your web pages.

    • HTML Images Syntax

    • <img src="url" alt="text">
    • The <img> tag is empty, it contains attributes only, and does not have a closing tag.
    • The <img> tag has two required attributes:
      • src - Specifies the path to the image
      • alt - Specifies an alternate text for the image
      • The width and height attributes always define the width and height of the image in pixels.

HTML Lists

HTML lists allow web developers to group a set of related items in lists.

  1. Unordered lists:
    • An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.
    • The list items will be marked with bullets (small black circles) by default
  2. Ordered lists:
    • An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.
    • The list items will be marked with numbers by default
  3. Description Lists:
    • A description list is a list of terms, with a description of each term.
    • The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag describes each term.
